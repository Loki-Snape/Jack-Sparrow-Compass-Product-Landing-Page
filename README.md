# ⚓️ Jack Sparrow's Compass — Product Landing Page

> "Not all treasure is silver and gold, mate."

Set sail with this whimsical, Pirates of the Caribbean–inspired product landing page. It's a lightweight static site built with simple HTML and CSS that showcases a fictional compass product for adventurous souls.

## What you'll find
- **Aesthetic:** Nautical, treasure-map vibes with bold headings and sea-salt styling.
- **Files:** Minimal project — `index.html` and `styles.css` are the entire ship's manifest.
- **Purpose:** Demo / landing page template ideal for playful product showcases.

## Preview locally
1. Open `index.html` directly in your browser (double-click), or run a simple HTTP server:

```bash
python3 -m http.server 8000
# then open http://localhost:8000 in your browser
```

2. For a quick preview in VS Code, press `Live Server` if you use the extension.

## Features to plunder
- Responsive hero section fit for a captain.
- Clean CSS with room to add parallax waves, animations, and treasure reveals.
- Easy to fork and brand for your own pirate wares.
